# Projeto Estacionamento Hexagonal Parte 1

A ideia dessa série é documentar o desenvolvimento de uma aplicação Simples,
mas aplicando DDD, Arquitetura Limpa e Hexagonal ou Ports & Adapters.

Dessa forma consigo fixar melhor e talvez ajudar alguém que vai chegar por aqui.

Como tecnologia utilizarei aspnet 8 para a API e React 18 para o frontend.

O foco é total no backend, mas assim não deixo meus conhecimentos de front enferrujarem
tanto.

## Escopo do Projeto

O cliente precisa ter o controle de entrada e saída do estacionamento e realizar
a cobrança.

Com as seguintes funcionalidades:

- Controlar entrada de carros (placa, horário)

- Controlar saída de carros (placa, horário)

- Calcular custo do estacionamento com base na entrada e saída ```(saida-entrada) *preco```

- Listar todos os carros Estacionados.

- Fechamento do dia Todos os carros
